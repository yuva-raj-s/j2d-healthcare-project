Run the Medallion Pipeline – Raw → Bronze → Silver → Gold
Each notebook transforms and validates data before passing to the next layer
15
Raw_NB – Ingest & Validate (Raw → Bronze)
Run ▶

What it does: Reads 3 CSVs from rawlayer, validates schema/nulls/types, writes validated Parquet to bronzelayer.

# Manual test run
input_date = "20260201"

# ADLS paths used
Source: rawlayer/Raw_Data/*.csv
Sink  : bronzelayer/Staging_Data/{dataset}/{input_date}/

Validation Checks Performed

  • Schema check – all expected columns present
  • Null check – critical fields (event_id, patient_id) not null
  • Type check – numeric columns are parseable as int
  • Exit value: "All validations are successfully passed"
  • Audit record written to J2D_Audit_table (layer=raw)
🛑If validation fails, Raw_NB exits with an error message and downstream notebooks will NOT run (fail-fast pattern).

Troubleshooting

IssueResolution
ADF Copy fails – MySQLVerify firewall allows Azure services; check username format j2dadmin@j2d-mysql-server
Databricks notebook fails at ADLS writeVerify spark.conf.set with correct account key or Key Vault secret scope
Synapse external table shows no rowsCheck Managed Identity has Storage Blob Data Reader on j2dstorage101
Parquet read error in SynapseEnsure Gold_NB wrote coalesce(1).write.format("parquet") not Delta
Power BI can't connectUse "Serverless SQL endpoint" not "Dedicated SQL endpoint" from Synapse overview
Logic App not firingConfirm Action Group is attached to Alert Rule; check Logic App run history
